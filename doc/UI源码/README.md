# UI Source Prototype Archive

This directory contains the pre-migration Next.js UI source used as a visual and interaction reference for the TanStack Start + Cloudflare implementation.

Treat this directory as read-only reference material:

- Do not add new product behavior here.
- Do not fix runtime issues here unless the prototype itself must be inspected.
- Implement the maintained application in `src/`.
- Keep this directory until the migrated UI has been stable in production long enough to remove the reference deliberately.
